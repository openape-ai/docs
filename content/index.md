# The security layer agents are missing.

AI agents are getting powerful. They send emails, move money, deploy code. OpenApe makes sure a human approves what matters — without slowing anything down.

## Agent Gatekeeping

::card-group
  ::card{title="grapes" icon="i-heroicons-shield-check" to="/ecosystem/grapes"}
  **The Grant System.** Request, approve, and manage permissions. Without a grant, nothing happens.
  ::
  ::card{title="shapes" icon="i-heroicons-cube" to="/ecosystem/shapes"}
  **The Execution Layer.** Run any CLI through structured permissions, constrained by grants and the Shapes Registry.
  ::
  ::card{title="escapes" icon="i-heroicons-bolt" to="/ecosystem/escapes"}
  **Privilege Escalation.** For when the agent needs to do something it should never normally be allowed to do.
  ::
::
